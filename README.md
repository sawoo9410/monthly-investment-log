# monthly-investment-log

개인 투자 기록 저장소. 월 단위로 매수/매도/보유 현황을 기록하고, 블로그 초안을 생성한다.

## 구조

```
.
├── CLAUDE.md           # Claude 작업 규칙
├── TEMPLATE/           # 월별 템플릿
│   ├── summary.md      # 블로그 초안 (5단 구조)
│   ├── transactions.md # 매매 내역
│   └── holdings.md     # 월말 보유 현황
├── scripts/
│   └── new-month.sh    # 월 폴더 생성
└── YYYY/YYYY-MM/       # 실제 기록
```

## 사용법

### 새 월 폴더 생성

```bash
./scripts/new-month.sh 2026-05
```

### 월간 정리

매수/매도 내역, 월말 보유 현황을 Claude에게 전달하며

> "이번 달 투자 내용을 정리하고 분석해"

라고 입력하면 해당 월 폴더의 모든 md 파일이 채워진다.

## 블로그 초안 구조

1. 저번 달 투자 근황
2. 이번 달 매수/매도 현황
3. 이번 달 보유 현황
4. 이번 달 매매 평가 (Claude 작성)
5. 뻘소리 (본인 작성, 항상 빈 상태)
