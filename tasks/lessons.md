# Lessons Learned — 교정 사항 누적

> 사용자로부터 교정받은 패턴을 기록한다.
> 세션 시작 시 반드시 이 파일을 검토하고 적용할 것.
> 단순 오타는 기록 불필요. **반복될 수 있는 패턴**만 기록.

## 데이터 처리

- 환율은 항상 사용자가 제공한 값을 사용할 것. 임의 환율 적용 금지.
- 현재가가 제공되지 않으면 평단 기준으로 산출하되, "확인 필요"에 명시.

## 포트폴리오 관찰 (누적)

- 사용자는 증권사별로 성격을 분리 운용: 토스(USD 개별주/ETF), 신한 ISA(국내 ETF + 현금), 한투(소량 국내 ETF).
- 신한 ISA의 현금(예수금) 비중이 높음 (2026-05 기준 43.4%) — 의도적인지 확인 필요.
