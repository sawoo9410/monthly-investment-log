# Monthly Todo Template

> 매월 작업 시 이 체크리스트를 참고한다. 실제 진행 상황은 여기에 기록하지 않고,
> 출력 form으로 사용자에게 보고한다.

---

## 월간 워크플로우 체크리스트

### Phase 1: 데이터 수집 & 검증
- [ ] 대상 월(YYYY-MM) 확인
- [ ] `tasks/lessons.md` 검토 — 적용할 교훈 확인
- [ ] 직전 달 `holdings.md` 존재 여부 확인
- [ ] 사용자 제공 데이터 완전성 점검:
  - [ ] 증권사별 매매 내역 (일자, 종목, 구분, 수량, 단가)
  - [ ] 월말 보유 현황 (종목, 수량, 평단, 현재가)
  - [ ] 환율 (USD 자산 있을 때)
- [ ] 누락 항목 판단: 작성 불가 → 질문 / 작성 가능 → 진행 후 "확인 필요"에 명시

### Phase 2: 파일 작성
- [ ] `./scripts/new-month.sh YYYY-MM` 실행 (폴더 없을 때)
- [ ] `transactions.md` 작성 (원본 가공 없이)
- [ ] `holdings.md` 작성 (증권사별 + 전체 합계)
- [ ] `summary.md` 작성 (5단 구조 엄수)
  - [ ] 1번: 전월 비교 or "직전 달 데이터 없음"
  - [ ] 2번: 매매 현황 표
  - [ ] 3번: 보유 현황 표 + 자산 비중
  - [ ] 4번: 매매 평가 (시장 상황 → 매매 평가 → 포트폴리오 코멘트)
  - [ ] 5번: 뻘소리 비워두기

### Phase 3: 검증
- [ ] 증권사별 소계 = 종목 금액 합계
- [ ] 전체 합계 = 증권사별 소계 합
- [ ] USD 환산: 금액(USD) × 환율 = 금액(KRW)
- [ ] transactions ↔ holdings 정합성
- [ ] 전월 대비 증감 정확성 (전월 데이터 있을 때)
- [ ] 5번 뻘소리 비어있음
- [ ] 종목명·수량·금액 = 사용자 원본

### Phase 4: 보고 & 기록
- [ ] 출력 form으로 사용자에게 보고
- [ ] 사용자 교정 발생 시 `tasks/lessons.md` 업데이트
- [ ] commit & push (사용자 요청 시)
